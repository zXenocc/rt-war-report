RT War Report (Real-Time War Report)
ระบบแจ้งเตือนและวิเคราะห์สถานการณ์ความขัดแย้งภูมิรัฐศาสตร์แบบเรียลไทม์ ผ่าน LINE โดยใช้พลังของ n8n และ AI Agent (Gemma 3)
📖 ภาพรวมของโปรเจค (Overview)
RT War Report เป็นเครื่องมือช่วยกรองข้อมูลข่าวสารที่ล้นทะลัก (Information Overload) เกี่ยวกับสงครามและความไม่สงบ โดยระบบจะดึงข้อมูลจากแหล่งข่าวที่เชื่อถือได้ (ACLED) มาทำการวิเคราะห์เชิงลึกด้วย AI สรุปประเด็นสำคัญ และส่งตรงถึงมือคุณผ่าน LINE ทันทีที่มีเหตุการณ์สำคัญเกิดขึ้น


✨ ฟีเจอร์เด่น (Key Features)
Dual-Trigger System: รองรับทั้งการส่งคำถามด้วยตัวเองผ่าน LINE (Webhook) และการตั้งเวลาดึงข้อมูลอัตโนมัติทุกชั่วโมง (Schedule)
Deep Analysis AI Agent: ใช้โมเดล Gemma 3 (via OpenRouter) ในบทบาทบรรณาธิการข่าวอาวุโส วิเคราะห์ทั้งในแง่มุมการทหาร การเมือง และเศรษฐกิจ
Smart Deduplication: มีระบบ Dedup Check ด้วย Code Node เพื่อป้องกันการส่งข่าวซ้ำซ้อนให้ผู้ใช้รำคาญ
Markdown Formatting: รายงานผลในรูปแบบที่อ่านง่าย มีหัวข้อชัดเจน และเป็นมืออาชีพ
