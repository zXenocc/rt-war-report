RT War Report (Real-Time War Report)
ระบบแจ้งเตือนและวิเคราะห์สถานการณ์ความขัดแย้งภูมิรัฐศาสตร์แบบเรียลไทม์ ผ่าน LINE โดยใช้พลังของ n8n และ AI Agent (Gemma 3)

📖 ภาพรวมของโปรเจค (Overview)
RT War Report เป็นเครื่องมือช่วยกรองข้อมูลข่าวสารที่ล้นทะลัก (Information Overload) เกี่ยวกับสงครามและความไม่สงบ โดยระบบจะดึงข้อมูลจากแหล่งข่าวที่เชื่อถือได้ (ACLED) มาทำการวิเคราะห์เชิงลึกด้วย AI สรุปประเด็นสำคัญ และส่งตรงถึงมือคุณผ่าน LINE ทันทีที่มีเหตุการณ์สำคัญเกิดขึ้น


🚀 ฟีเจอร์เด่นในเวอร์ชัน
Dual-Trigger Intelligence: รองรับการทำงานทั้งแบบตั้งเวลา (Schedule) ทุก 1 ชั่วโมง และแบบตอบโต้ทันทีเมื่อผู้ใช้ทักแชท (Webhook).
AI Agent with Memory: ใช้โมเดล GPT-4o-mini (ผ่าน OpenRouter) ร่วมกับระบบ Window Buffer Memory เพื่อให้ AI จำบริบทการสนทนาได้.
Hybrid Notification: แยกการส่งข้อความอัตโนมัติ:
Reply Mode: ตอบกลับเฉพาะบุคคลด้วย replyToken เมื่อมีการสอบถาม.
Broadcast Mode: ประกาศข่าวใหม่ให้ผู้ใช้ทุกคนทราบเมื่อตรวจพบเหตุการณ์สำคัญจากระบบ Schedule.
Advanced Deduplication: ใช้ Code Node (JavaScript) เพื่อเปรียบเทียบข้อมูลใหม่กับ staticData ป้องกันการแจ้งเตือนข่าวเดิมซ้ำซ้อน.
